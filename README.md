# Display Backend Usage

## Install Dependencies
```sh
composer install
```

## Import Data
```sh
php bin/console ugo:orders:import
```

## Start Server
```sh
symfony serve
```
default url is usually ```http://127.0.0.1:8000 ```


## Execute Tests
```sh
php bin/phpunit
```
